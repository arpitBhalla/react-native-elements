# @rneui/envinfo

Logs information about the current environment relevant to packages to the console.

## Usage

```bash
npx @rneui/envinfo
```

## Output

```md
## Global Dependencies:

- @rneui/envinfo@1.0.0
- expo-cli@5.3.1

## Local Dependencies:

- @rneui/base@^4.0.0-rc.3
- @rneui/layout@^0.0.0-alpha.3
- @rneui/themed@latest
- react-native-elements@^3.4.2
```
