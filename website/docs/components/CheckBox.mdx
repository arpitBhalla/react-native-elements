CheckBoxes allow users to complete tasks that involve making choices such as selecting options, or switching settings - On or Off.
It provides a clear visual of either a true or false choice.

## Props

<div class='table-responsive'>

| Name              | Type                                                                   | Default           | Description                                                               |
| ----------------- | ---------------------------------------------------------------------- | ----------------- | ------------------------------------------------------------------------- |
| `Component`       | React Component                                                        |                   | Specify React Native component for main button.                           |
| `center`          | boolean                                                                | `false`           | Aligns checkbox to center.                                                |
| `checked`         | boolean                                                                | `false`           | Flag for checking the icon.                                               |
| `checkedColor`    | string                                                                 | `Color [Primary]` | Default checked color.                                                    |
| `checkedIcon`     | `string` or `ReactElement<{}, string` or `JSXElementConstructor<any>>` |                   | Default checked icon ([Font Awesome Icon](http://fontawesome.io/icons/)). |
| `checkedTitle`    | string                                                                 |                   | Specify a custom checked message.                                         |
| `containerStyle`  | View style(Object)                                                     |                   | Style of main container.                                                  |
| `fontFamily`      | string                                                                 |                   | Specify different font family.                                            |
| `iconRight`       | boolean                                                                | `false`           | Moves icon to right of text.                                              |
| `iconType`        | string                                                                 |                   | Type of icon set. [Supported sets here](icon#available-icon-sets).        |
| `onIconPress`     | Function                                                               |                   | onPress function for checkbox.                                            |
| `onLongIconPress` | Function                                                               |                   | onLongPress function for checkbox.                                        |
| `right`           | boolean                                                                | `false`           | Aligns checkbox to right.                                                 |
| `size`            | number                                                                 |                   | Size of the checkbox.                                                     |
| `textStyle`       | Text Style(Object)                                                     |                   | Style of text.                                                            |
| `title`           | `string` or `ReactElement<{}, string` or `JSXElementConstructor<any>>` |                   | Title of checkbox.                                                        |
| `titleProps`      | TextProps                                                              | `{}`              | Additional props for the title Text component.                            |
| `uncheckedColor`  | string                                                                 |                   | Default unchecked color.                                                  |
| `uncheckedIcon`   | `string` or `ReactElement<{}, string` or `JSXElementConstructor<any>>` |                   | Default checked icon ([Font Awesome Icon](http://fontawesome.io/icons/))  |
| `wrapperStyle`    | View style(Object)                                                     |                   | Style for the wrapper of checkbox.                                        |
