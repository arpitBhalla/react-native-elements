This is used for adding an accessory to the Avatar.
Receives either all [Icon](icon#props) or [Image](image#props) props.

## Usage

## Props

:::note
Include all props from IconButtonProps, TouchableHighlightProps
:::

<div class='table-responsive'>

| Name             | Type                                 | Default | Description                                                                                                                                                                        |
| ---------------- | ------------------------------------ | ------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `pressableProps` | PressableProps except click handlers | `None`  |                                                                                                                                                                                    |
| `style`          | Text Style(Object)                   |         | Style prop inherited from TextProps and TouchableWithoutFeedbackPropertiesOnly exist here so we can have ViewStyle or TextStyleStyleAdd custom styling to the accessory of avatar. |
| `underlayColor`  | ColorValue                           | `#000`  | The color of the underlay that will show through when the touch is active.Add underlay color to the accessory of avatar.                                                           |
